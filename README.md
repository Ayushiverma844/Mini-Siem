# MINI SIEM

A Machine Learning based Mini SIEM system that detects phishing and suspicious log activities.

## Features

- Log monitoring
- Phishing detection
- Machine learning model
- Web interface

## Technologies Used

- Python
- Flask
- Scikit-learn
- Pandas
- HTML / CSS / JavaScript

## How to Run

Install dependencies:

pip install -r requirements.txt

Run:

python app.py
